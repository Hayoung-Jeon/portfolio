회원가입 유효성 검사

1. 폼에서 사용자의 정보를 입력받을 때, 입력받은 정보가 요구하는 항목에 적합한 형식인지 판가름하도록 설계했습니다.
2. 정규식과 if-else문을 활용해 각 항목에 대한 유효성 검사를 실행하도록 했습니다.
3. 키 입력 시 마다 유효성 검사를 실시하고, 미입력한 채 넘어가는 것을 확인하기 위해 포커스를 잃을 때 폼에 값이 존재하는지 확인하도록 설정했습니다.
4. 각 항목별로 설정된 조건은 입력폼 상단에 명시되어 있습니다.
  • 1. 알파벳 혹은 숫자로 구성되어 있을 것
    2. 사이에 @ 표시가 들어갈 것
  • 한글 이름만 입력받을 것 (숫자, 알파벳 X)
  • 1. 10자 - 20자 사이일 것
    2. 숫자, 대문자, 특수문자 중 2개 이상 사용
  • 비밀번호와 일치할 것
  • - 포함한 숫자로만 구성
  • YYYYMMDD 형식으로 년 → 월 → 일 순으로 입력
